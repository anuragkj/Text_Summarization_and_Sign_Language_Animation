This example shows how individual elements of the GUI can be created directly.

The avatar panel is generated automatically, but other buttons and fields are provided explicitly. The class name on each element identifies the type of element and the number of the avatar. Multiple instances of buttons and output fields can be used, but input fields such as the SiGML URL or SiGML Text should occur only once for a given avatar.
