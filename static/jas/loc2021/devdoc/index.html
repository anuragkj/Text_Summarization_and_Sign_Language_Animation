JASigning Developer Documentation

As well as supporting avatar definitions, configuration data, etc., JASigning consists of three Java modules and a native library:

  • sigmlinlib (Java)
    Input and translation of SiGML data;
  • jarp (Java)
    Basic avatar animation, and signing avatar management;
  • use-jarp (Java)
    Standard apps and applets for jarpb;
  • jaglib (animgen native library)
    Generates avatar animation data from SiGML.

In addition to the standard Java 2 platform, JASigning depends on three externally developed libraries: ANTLR (parser generation library), JOGL (OpenGL interface library), and expat (XML parsing library).

The currently available documentation can be downloaded as ZIP archives:

  • sigmlinlib javadoc
  • jarpb javadoc
  • use-jarp javadoc
  • use-jarp source code.

A version of JASigning is available for installation on a localhost server. The installation includes a NetBeans project for building freestanding apps using use-jarp. See JASigning.
